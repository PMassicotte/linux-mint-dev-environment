Setting up my terminal environment. The dimensions of my terminal is 150 columns by 35 rows.

1.1 Fonts

1.2 zsh

sudo apt install zsh

1.2.1 oh-my-zsh

Install oh-my-zsh

sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

1.2.2 power10k theme

Powerlevel10k theme

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

Then, set ZSH_THEME="powerlevel10k/powerlevel10k" in ~/.zshrc and configure the theme as follow:

p10k configure

1.3 Useful tools

  • Useful tools written in rust

For some of these tools, rust needs to be installed.

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

1.3.1 tldr

sudo apt install tldr
tldr --update

1.3.2 exa

cargo install exa

1.3.3 bat

cargo install bat

1.3.4 Creating alias

nano ~/.zshrc

Add these alias to the file.

alias ls='exa --icons'
alias l='exa'
alias la='exa -a'
alias ll='exa -lahF --icons'
alias cat=bat

1.3.5 fzf

fzf is a command-line fuzzy finder.

git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install

Then, set plugins=(fzf) in ~/.zshrc.

1.3.6 The Silver Searcher

The Silver Searcher

sudo apt-get install silversearcher-ag

1.3.7 ripgrep

https://github.com/BurntSushi/ripgrep

Easier to use to me.

rg read.csv -g "*.R"

1.3.8 gdu - go DiskUsage()

Pretty fast disk usage analyzer written in Go: https://github.com/dundee/gdu

add-apt-repository ppa:daniel-milde/gdu
apt-get update
apt-get install gdu

1.4 Terminator

I am using Terminator as my terminal emulator.

  • Installing themes

I am using this following TOML config file:

nvim ~/.config/terminator/config

  title_transmit_fg_color = "#000000"
  title_transmit_bg_color = "#fabd2f"
  enabled_plugins = LaunchpadBugURLHandler, LaunchpadCodeURLHandler, APTURLHandler, TerminatorThemes
  suppress_multiple_term_dialog = True
  title_use_system_font = False
  title_font = MesloLGS NF Bold 12
  always_split_with_profile = True
[keybindings]
[profiles]
  [[default]]
    cursor_color = "#aaaaaa"
    font = MesloLGS NF Bold 10
    use_system_font = False
  [[Nord]]
